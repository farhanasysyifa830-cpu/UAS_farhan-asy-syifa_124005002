# Catch A Monster Joki

Website Jasa Joki Game Catch A Monster

## Fitur

- Login Admin & Customer
- Register
- Dashboard
- CRUD Order
- Riwayat Order
- Chat Admin
- Pengaturan Website

## Teknologi

- PHP Native
- MySQL
- Bootstrap 5

## Cara Menjalankan

1. Import database `database/db_joki.sql`
2. Pindahkan project ke folder `htdocs`
3. Jalankan XAMPP (Apache & MySQL)
4. Buka `http://localhost/nama-folder`
